# Building Energy Consumption Prediction Using Machine Learning

This project applies machine learning techniques to predict building energy consumption based on environmental and operational features. The objective is to compare multiple regression models and evaluate their performance in predicting energy usage.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature scaling
- Model training and comparison
- Performance evaluation using MAE and MSE
- Desktop GUI developed with Tkinter

## Machine Learning Models

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
- Tkinter

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
